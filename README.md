#  Planova — Smart Event Planning & Booking Platform

**Planova** is a simple and reliable platform designed to make event planning and ticket booking effortless for everyone.

Whether you're organizing an event or looking to attend one, Planova brings everything together in one place.

---

##  What Planova Does

Planova helps people:

- Discover upcoming events
- Book tickets بسهولة (quickly and securely)
- Manage their bookings
- Organize and publish events

It connects **event organizers** with **attendees**, making the entire experience smooth and efficient.

---

##  Who Is It For?

###  Attendees

Anyone who wants to:

- Browse available events
- Reserve a spot
- Keep track of bookings

###  Event Organizers

People or organizations who want to:

- Create and publish events
- Control how many people can attend
- Monitor bookings and attendance

---

##  Key Features

###  Discover Events

Users can easily explore different events happening around them, including details like date, description, and availability.

---

###  Easy Ticket Booking

Booking a ticket is quick and straightforward. Once booked, your spot is secured.

---

###  No Overbooking

Planova ensures that events never exceed their capacity.
If an event is full, no additional bookings are allowed.

---

###  Booking Management

Users can:

- View their bookings
- Cancel if plans change

---

###  Event Management

Organizers can:

- Create new events
- Set attendance limits
- Manage all bookings in one place

---

##  How It Works

1. **Browse Events**
   Users explore available events.

2. **Book a Ticket**
   Reserve a spot instantly.

3. **Event Capacity Check**
   Planova ensures the event is not already full.

4. **Manage Booking**
   Users can cancel if needed.

5. **Organizer Control**
   Event creators monitor and manage attendance.
